# Flaxen Wix Website Mockup

Static ecommerce-style mockup for Flaxen Wix candles.

## Files

- `index.html` - full homepage / shop mockup
- `style.css` - site styling
- `script.js` - mock cart and email order enquiry
- `images/` - image files used by the site
